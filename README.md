# 한국경제 뉴스 모음

이 프로젝트는 주요 경제 뉴스를 자동으로 수집하여 웹페이지로 제공합니다.

## 실시간 뉴스 보기

아래 링크에서 최신 경제 뉴스를 확인할 수 있습니다:
https://[YOUR_GITHUB_USERNAME].github.io/korean-economy-news/

## 로컬에서 실행하기

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. 스크립트 실행:
```bash
python korean_economy_news.py
```

## 기능

- 주요 경제 신문사의 최신 뉴스 수집
- 경제, 환율, 금리 등 주요 키워드 기반 필터링
- 깔끔한 웹 인터페이스
- 매일 자동 업데이트
